Vite Starters

We’ve created a collection of Vite starters to help you quickly scaffold and explore Corex integrations using Vanilla JavaScript and TypeScript. These templates are organized around different design layers (Corex, Modex, Themex), with CSS or Tailwind options — both in their default and customizable forms.
