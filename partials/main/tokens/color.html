Color

A flexible and accessible color palette management system. Supports multiple palette colors, customizable scales, and both light and dark modes.

Introduction

The Corex color system is designed to be both accessible and flexible. It adapts to your design needs without locking you into a specific palette or scale.

There’s no configuration or external dependencies required—your color tokens are the single source of truth.

Each Corex template can use a different color palette and even a different scale by adapting the same set of semantic tokens. This makes the system both consistent and highly customizable across projects.

Stay tuned — we’re working on new templates that showcase unique combinations of color strategies, helping you explore even more design possibilities with Corex.
